# Dotfiles

My configuration files for OSX and Ubuntu.

## Install

Just copy-paste all and hit enter.
```sh
git clone https://pauloromeira@github.com/pauloromeira/dotfiles.git \
~/.dotfiles && ~/.dotfiles/bootstrap.sh
```
